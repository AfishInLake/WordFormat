# 常用字号对照表

WordFormat 配置文件中 `font_size` 字段支持中文字号名称和磅值两种格式。

## 中文字号 → 磅值对照

| 字号 | 磅值 | 用途场景 |
|------|------|----------|
| 初号 | 42磅 | 封面标题 |
| 小初 | 36磅 | 封面副标题 |
| 一号 | 26磅 | 章节大标题 |
| 小一 | 24磅 | — |
| 二号 | 22磅 | — |
| 小二 | 18磅 | 本科一级标题、摘要标题 |
| 三号 | 16磅 | 本科二级标题、参考文献标题 |
| 小三 | 15磅 | 硕士英文摘要标题 |
| 四号 | 14磅 | 本科英文摘要标题 |
| 小四 | 12磅 | **正文（最常用）** |
| 五号 | 10.5磅 | 图表题注、参考文献内容 |
| 小五 | 9磅 | 期刊摘要正文、脚注 |
| 六号 | 7.5磅 | — |
| 小六 | 6.5磅 | — |
| 七号 | 5.5磅 | — |
| 八号 | 5磅 | — |

## 常见论文格式字号搭配

### 本科毕业论文（典型）
- 一级标题：小二（18磅）
- 二级标题：三号（16磅）
- 三级标题：小四（12磅）
- 正文：小四（12磅）
- 摘要标题：小二（18磅）
- 英文摘要标题：四号（14磅）
- 关键词：小四（12磅）
- 图表题注：五号（10.5磅）
- 参考文献内容：五号（10.5磅）

### 硕士毕业论文（典型）
- 一级标题：三号（16磅）
- 二级标题：四号（14磅）
- 三级标题：小四（12磅）
- 正文：小四（12磅）
- 摘要标题：三号（16磅）
- 英文摘要标题：小三（15磅）

### 期刊论文（典型）
- 一级标题：三号（16磅）加粗
- 二级标题：四号（14磅）加粗
- 正文：五号（10.5磅）
- 摘要：小五（9磅）
